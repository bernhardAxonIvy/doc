.. _cms-editor:

CMS Editor
==========

Use the CMS Editor to add, delete, and edit the Content Objects of your project.
Select a Content Object in the table to view and edit its values.

.. figure:: /_images/cms-editor/cms-editor.png
   :alt: CMS Editor
  
   CMS Editor

Tools
-----

|add-icon| Add Content Object: Add a new Content Object to your project.

.. |add-icon| image:: /_images/ui-icons/plus.svg
   :width: 2em

.. figure:: /_images/cms-editor/cms-editor-add-content-object.png
   :alt: Add Content Object
  
   Add Content Object


|delete-icon| Delete: Delete the selected Content Objects from your project.

.. |delete-icon| image:: /_images/ui-icons/trash.svg
   :width: 2em

|language-icon| Language Tools: Open a drop down menu containing the available
Language Tools.

.. |language-icon| image:: /_images/ui-icons/language.svg
   :width: 2em

Language Tools
~~~~~~~~~~~~~~

|language-manager-icon| Language Manager: Add or remove languages for your
project.

.. |language-manager-icon| image:: /_images/ui-icons/world-cog.svg
   :width: 2em

.. figure:: /_images/cms-editor/cms-editor-language-manager.png
   :alt: Language Manager
  
   Language Manager

.. _translation-wizard:

|language-icon| Translation Wizard: Quickly fill values of the selected Content
Objects with machine translations.

.. figure:: /_images/cms-editor/cms-editor-translation-wizard.png
   :alt: Translation Wizard
  
   Translation Wizard

.. note::

   To use the Translation Wizard, the :ref:`translation-service` must be
   configured on the Axon Ivy Engine.
