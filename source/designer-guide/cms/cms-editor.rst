.. _cms-editor:

CMS Editor
==========

Use the CMS Editor to add, delete, and edit the Content Objects of your project.
Select a Content Object in the table to view and edit its properties.

.. figure:: /_images/cms-editor/cms-editor.png
   :alt: CMS Editor
  
   CMS Editor

Language Tools
--------------

Language Manager
~~~~~~~~~~~~~~~~

Use the Language Manager to add or remove languages for your project.

.. figure:: /_images/cms-editor/cms-editor-language-manager.png
   :alt: Language Manager
  
   Language Manager

.. _translation-wizard:

Translation Wizard
~~~~~~~~~~~~~~~~~~

Use the Translation Wizard to fill values of selected Content Objects with
machine translations.

.. figure:: /_images/cms-editor/cms-editor-translation-wizard.png
   :alt: Translation Wizard
  
   Translation Wizard


.. note::

   To use the Translation Wizard, the :ref:`translation-service` must be
   configured on the Axon Ivy Engine.
